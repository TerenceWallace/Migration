GridSearch GridCircleAround Method Migration Documented Class Library
Circles around the given center while maintaining a constant distance of inRadius to the center and a distance of inStepWidth to the previous call of the handler, if possible. In contrast to WalkAround, this method does NOT cover an area but only a closed line (approx. a circle).
