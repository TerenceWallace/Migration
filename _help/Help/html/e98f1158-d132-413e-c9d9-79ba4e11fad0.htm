SetPixels(Bitmap)
SetPixels(Bitmap,  Byte )
If image is already compressed, the following method will load it directly, omitting any unneccessary internal postprocessing in the OpenGL implementation/driver. But since compression is system dependent, you must also provide the original image in case the given compression is not supported. If this is the case, the method will behave exactly as SetPixel.
