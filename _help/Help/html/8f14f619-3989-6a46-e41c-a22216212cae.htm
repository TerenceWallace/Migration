TerrainDefinition AverageDrawToHeightmap Method Migration Documented Class Library
Instead of DrawToHeightmap(Int32, Int32, TerrainBrush), this one will use the brush to reduce the terrain variance from average. This will in fact grade the terrain to the given average height by using the brush as weight for each surrounding cell.
