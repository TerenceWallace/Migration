ResourceManager AddResourceStack Method (CyclePoint, StackType, Resource, Int32) Migration Documented Class Library
Adds a new resource stack with the given parameters and raises OnAddStack. This method is also called internally and it is crucial to do all postprocessings meant to be applied for all resource stacks, like attaching a visual for rendering, in the mentioned event.
