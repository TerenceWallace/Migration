SetPath(Movable, Point, Procedure Boolean )
Sets a path with an optional result handler. It is not guaranteed that a path can be found, neither that it will be reached. What is guaranteed is that the result handler is called in any case and only with "true" as parameter if the movable has reached its destination. Further, a movable with a pending result handler is considered to be non-idle. If you try to set a path for a movable with a pending result handler ("doing a job"), the call will immediately throw an exception!
